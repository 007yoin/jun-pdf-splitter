# Jun-PDF-Splitter

PDF 파일을 간편하게 분할하고 병합할 수 있는 Windows 프로그램입니다.

## 다운로드

[Releases](https://github.com/007yoin/jun-pdf-splitter/releases) 페이지에서 최신 버전을 다운로드하세요.

## 사용법

### PDF 분할

1. "분할" 탭을 선택합니다
2. PDF 파일을 선택하거나 드래그 앤 드롭합니다
3. 저장할 폴더를 선택합니다
4. 챕터를 추가합니다:
   - 챕터 이름 입력
   - 시작 페이지와 끝 페이지 입력
   - "챕터 추가" 버튼 클릭 (또는 Enter 키)
5. 필요한 만큼 챕터를 추가한 후 "분할 실행" 버튼을 클릭합니다

### PDF 병합

1. "병합" 탭을 선택합니다
2. 병합할 PDF 파일들을 추가합니다 (드래그 앤 드롭 가능)
3. 위/아래 버튼으로 순서를 조정합니다
4. 저장할 파일 경로를 지정합니다
5. "병합 실행" 버튼을 클릭합니다

## 시스템 요구사항

- Windows 10 이상
- 64비트 운영체제
